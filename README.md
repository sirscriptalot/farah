# Farah

Farah is a tiny library for implementing "runners" (service objects).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'farah'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install farah

For RubyMotion, require the gem inside your Gemfile:

```ruby
require 'farah'
```
